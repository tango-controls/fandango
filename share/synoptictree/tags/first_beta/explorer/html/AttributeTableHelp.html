Attribute table

The attribute table consists of a table with attributes. The attributes come in three formats, scalars, spectrums, and images. The value of the attributes are shown in separate tabs for the spectrums and the images, whereas the scalar-values are shown directly in the table.
The device field is used to indicate the state of the corresponding device with predefined colours and the state of the device is also available as a tool-tip. The quality of the attributes are available as colours on the name-field, and the name of the quality is available as a tool-tip.
The rows in the table can be selected and when selected and right-clicked a popup-menu will appear with the following items:
  • Remove - removes the current attribute. This item is only available in administrator-mode
  • Remove all - removes all attributes from the table. This item is only available in administrator-mode
  • View - which lets you choose which fields to view.
  • Status - which opens a window with the status of the corresponding device
  • Run Jive - which lets you start Jive, a program used to configure devices. This item is only enabled if jive is available to the Device Tree
  • Run ATKpanel - which lets you start ATKPanel on the device of this attribute. This item is only enabled if ATKPanel is available to the Device Tree.
  • Refresh - which lets you refresh this attribute.
  • Set - which lets you set the value of the attribute. This item is only enabled if the attribute is of type write.
  • Trend - which lets you open a trend window to see the values of the attribute plotted against time. This item is only enabled if the attribute is of type numberscalar.
  • Add to trend - which let you add this attribute to a trendwindow with other attributes, so that they can be plotted against each others.
